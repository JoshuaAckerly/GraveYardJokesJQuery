The purpose of this site is to display my skills in web development and the use of different plugins and JavaScript libraries. In the home page I have a jQuery navigation menu that will have pop up menus that come from a list of elements pre coded in the design. From the home page, and each page I have set up, you are able to navigate to each of the pages. I used different libraries such as jQuery Ui to create some animations that a user will be able to click a button, or click on to perform any functionality that is present.

I used the fonts that I did because I believe they made sense for the overall theme of the website. A professional clean looking site that is user friendly and uses blues to convey a tone of honesty. The website features different images that help to tell the story of what the site is about and help to enhance the experience of the user.
